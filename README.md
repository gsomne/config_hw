# Учебный конфигурационный язык → YAML

## Описание

Данный проект представляет собой инструмент командной строки, который преобразует текст,
написанный на учебном конфигурационном языке, в формат YAML.

Входные данные читаются из стандартного ввода (stdin). Результат трансляции записывается
в YAML-файл, путь к которому задаётся ключом командной строки. Все синтаксические ошибки
выявляются и выводятся в стандартный поток ошибок.

Проект реализован на языке Python.

## Структура проекта
``` text
.
├── main.py
├── input.conf
└── README.md
```
При обращении к необъявленной константе возникает синтаксическая ошибка.

## Соответствие конструкций YAML

| Конструкция учебного языка | Представление в YAML |
|---------------------------|----------------------|
| Число                     | число                |
| Строка                    | строка               |
| list                      | список               |
| struct                    | словарь              |
| set                       | не выводится         |

## Использование
```bash
python main.py -o output.yaml < input.conf
```

Где:
- input.conf — файл с конфигурацией на учебном языке
- output.yaml — файл с результатом трансляции в формате YAML

## Пример входного файла

```text
–[[
Конфигурация сервера
]]

set port = 8080.0

struct {
host = ‘localhost’,
ports = (list |port| 443.0),
}
```

## Пример выходного файла

```yaml
host: localhost
ports:
- 8080.0
- 443.0
```

## Реализованные возможности

	•	лексический анализ входного текста
	•	синтаксический анализ методом рекурсивного спуска
	•	поддержка вложенных массивов и структур
	•	вычисление констант на этапе трансляции
	•	игнорирование многострочных комментариев
	•	формирование корректного YAML-файла

Требования
	•	Python 3.9 или выше
	•	библиотека PyYAML
